# Rap Express Backlog

Epics map directly to PLAN.md phases. Each epic folder contains user-story markdown files describing scope, acceptance criteria, dependencies, and deliverables.

## Epic Index
1. `epic-foundations` – repo setup, CMS schema, design system.
2. `epic-ux-content` – IA, copy, photography, SEO groundwork.
3. `epic-frontend` – page templates, components, quote forms, coverage map.
4. `epic-backend` – APIs, database, integrations (CRM, payments, notifications).
5. `epic-launch` – QA, A11y, SEO polish, content population, go-live.
6. `epic-postlaunch` – automation, city landing pages, testimonial sync, portal enhancements.
