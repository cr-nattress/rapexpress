# US-052: API Docs & Client Dashboard Enhancements

**Epic:** Post-Launch Enhancements

## Description
- Publish API documentation (or webhook guide) for enterprise clients wanting automation.
- If building custom portal layer, add dashboard view (recent deliveries, status, invoice download).

## Acceptance Criteria
- API docs accessible (Swagger/Redoc or MDX) with authentication instructions.
- Portal MVP allows clients to log in and view at least recent quote submissions + statuses.

## Dependencies
- Backend endpoints stable, auth decision made.
