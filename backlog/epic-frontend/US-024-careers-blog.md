# US-024: Careers & Blog Templates

**Epic:** Frontend

## Description
- Implement careers page with application upload form, culture content, team bios.
- Build blog listing + article templates referencing CMS content.

## Acceptance Criteria
- Applicants can upload PDF/doc (with file size validation) and receive confirmation.
- Blog pages render SEO metadata, social cards, table of contents (if long form).
- Careers page includes embed to download application & notary requirement.

## Dependencies
- CMS entries, backend upload endpoint.
